# Project Classified

<Callout title="🔴 Access Required">
You need a valid **access code** to enter the Classified vault. [Join the waitlist](/agent) on the Agent page — your code arrives in your inbox instantly.
</Callout>

Project Classified is PyVax's gated initiative for competitive agent building. Designed as a cyberpunk portal for early-access members, it reveals everything you need to build and deploy autonomous agents. Currently, we are supporting developers entering the **Synthesis Hackathon** — a live on-chain competition with a **$75,000 prize pool**.

## What Is It?

When you sign up for early access via the [Agent page](/agent), you receive a **unique access code** by email. This code acts as your key to the vault at `/classified`. Inside, you'll find:

- **The full hackathon brief** — mission, rules, submission criteria
- **Prize breakdown** — $75K distributed across competition tracks
- **Step-by-step setup** — how to install `classified-agent`, authenticate, and submit
- **Terminal demo** — a live preview of an autonomous agent executing on Avalanche C-Chain
- **Registration link** — official Synthesis Hackathon registration on Unstop

## The Aesthetic

The Classified portal is intentionally designed to feel covert: deep purple-black backgrounds, hot-pink glows, redacted text blocks, and a particle field backdrop. It loads with a cinematic loading screen followed by an access gate.

## Access Flow

```
Sign up at /agent
     ↓
Receive access code by email (e.g. A3F9B2C1)
     ↓
Navigate to /classified
     ↓
Enter your code at the access gate
     ↓
Full portal unlocks (session stored for 30 days)
```

The code is stored in your browser's `sessionStorage` so you don't need to enter it again during the same session. Across devices or sessions up to 30 days, re-entering your code instantly re-authenticates.

## Synthesis Hackathon

The Synthesis Hackathon is a major external competition where Project Classified developers can test their agents live. Participants build autonomous Python agents using the PyVax SDK and deploy them to Avalanche C-Chain. Agents are judged on:

- **Functionality** — does the agent accomplish its mission autonomously?
- **Efficiency** — gas optimization and execution finality
- **Innovation** — novel use of `@agent_action`, `@human_action`, agent memory, and multi-agent coordination
